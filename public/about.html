How to Use:
  1. Open the Editor
  2. Search for Albums by Artist

    3. Click the album image to open

  3. Add desired songs

    4. Added songs will appear at the top the editor window for viewing and deleting

  4. After all desired songs are added, goto "My Playlist"

    5. Select the row of the song you'd like to move and use the arrow buttons to change its order

  5. When ready, click "Save" to save the playlist to the server.

    6. Your playlist is then written to the server's storage file
    If the playlist is not saved, the browser's local storage will still contain the most recent playlist for future editing

  6. Click "Get Playlist" to retrieve the last saved playlist from the server

    7. The playlist is returned in JSON format with complete track info for each song
